# Microcloud
一个基于Java SE的多线程 ,可断点下载的URL资源下载器(Windows桌面版)
可跟另一个云盘项目MicroClouds(web版)配合管理上传/下载的文件 . (未完成)
核心功能(上传/下载/多线程/断点)已经完成,界面UI交换没有跟进.
